Each FTIR spectrum needs information about the sample thickness and the file location¶

I prefer to first specify a sample and keep sample thickness information there

File location is indicated by the folder, which you specify, and a label ('fname', which is also required input for plotting titles and saving data later). In this example the file with the FTIR spectra information is pynamsDemo.CSV

In [115]:
# Make sample with thickness and optional other info
sample1D = pynams.Sample()
sample1D.sample_thick_microns = np.mean([885., 867., 873., 878., 879.])
sample1D.IGSN = 'IEFERPMR1'

# Make spectrum, including sample info and filename
spectrum = pynams.Spectrum()
spectrum.sample = sample1D # specify sample
FTIR_file_location = 'C:\\Users\\Ferriss\\Documents\\FTIR\\'
spectrum.fname = 'pynamsDemo'
pynams.make_filenames(folder=FTIR_file_location, file_ending='.CSV')  # only .CSV files supported for now

fig, ax = spectrum.plot_spectrum()
